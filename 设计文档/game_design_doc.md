> 开发方案 | 单人独立游戏 | Unity 2D 俯视角 Roguelite 搜打撤

## 一、项目概述

### 1.1 游戏简介

**《撤离者》** 是一款 2D 俯视角 Roguelite 搜打撤游戏。玩家选择起始特质进入程序生成的地图，在探索中不断获得随机强化、遗物和装备，构建出独特的build。最终找到撤离点成功撤出，带回来的核心遗物可以永久解锁，成为后续局的优势。

### 1.2 一句话定位

每局都是不同的build体验，撤出来才算你的。

### 1.3 核心循环

```
选择特质 → 进入地图 → 局内成长（升级/遗物/事件）→ 撤离
                    ↓
              撤离成功 → 解锁永久遗物/解锁新特质
              撤离失败 → 本局遗物丢失
```

### 1.4 基本信息

| 项目 | 内容 |

## 二、核心系统设计

### 2.1 Roguelite 核心系统（新增）

#### 2.1.1 起始特质系统

每局开始前，从 3 个随机特质中选择 1 个作为本局起点。

**特质类型：**

| 特质名称 | 效果 | 玩法倾向 |
|---------|------|---------|
| 重装士兵 | +30%生命，-10%移速 | 坦克流 |
| 疾风猎手 | +20%移速，+10%暴击 | 速通流 |
| 弹药专家 | 弹药+50%，换弹-30% | 火力流 |
| 拾荒者 | 搜刮速度+50%，背包+2格 | 搜刮流 |
| 近战狂人 | 近战伤害+50%，远程-20% | 近战流 |
| 幸运儿 | 稀有掉落率×2，伤害-10% | 运气流 |
| 老兵 | 起始武器已强化+1，无特殊 | 稳定流 |

**解锁机制：**
- 6个特质初始锁定
- 完成特定成就解锁（如"用近战击杀50敌人"解锁近战狂人）

#### 2.1.2 局内升级系统

击杀敌人获得经验，升级时从 3 个随机选项中选择 1 个强化。

**升级选项池：**

| 类别 | 选项示例 | 效果 |
|------|---------|------|
| 攻击 | 穿透弹 | 子弹穿透1个敌人 |
| 攻击 | 散射 | 子弹数量+2，伤害-20% |
| 攻击 | 暴击专精 | 暴击率+15%，暴击伤害+50% |
| 攻击 | 燃烧弹 | 命中后造成持续燃烧伤害 |
| 防御 | 铁壁 | 受到伤害-15% |
| 防御 | 闪避大师 | 闪避率+10% |
| 防御 | 吸血 | 击杀回复5HP |
| 防御 | 护盾 | 每次进房间获得20点临时护盾 |
| 机动 | 冲刺强化 | 冲刺冷却-30% |
| 机动 | 影步 | 冲刺后1秒无敌 |
| 机动 | 疾跑 | 移速+20% |
| 辅助 | 搜刮专家 | 搜刮速度+40% |
| 辅助 | 鹰眼 | 显示相邻房间敌人位置 |
| 辅助 | 弹药回收 | 击杀有30%概率掉落弹药 |

**升级权重：**
- 每局根据已选特质和遗物动态调整权重
- 避免出现与当前build冲突的选项

#### 2.1.3 遗物系统

遗物是改变玩法的核心道具，每局随机获得。

**遗物分类：**

| 稀有度 | 颜色 | 获取方式 | 示例 |
|-------|------|---------|------|
| 普通 | 白色 | 房间奖励、普通敌人 | 弹药+20%、移速+10% |
| 精良 | 蓝色 | 精英怪、事件奖励 | 暴击穿透、杀敌回血 |
| 稀有 | 紫色 | Boss、特殊事件 | 双持武器、子弹弹射 |
| 传说 | 金色 | 隐藏房间、完美撤离 | 死亡复活1次、伤害翻倍但生命减半 |

**遗物设计原则：**
- 每个遗物都显著改变玩法，而非简单数值堆叠
- 遗物之间可以组合出强力build
- 某些遗物有负面效果，换取更强的正面效果

**遗物示例：**

| 遗物名称 | 稀有度 | 效果 |
|---------|-------|------|
| 贪婪者契约 | 普通 | 金币+50%，但商店物品价格×1.5 |
| 铁匠锤 | 普通 | 武器伤害+10% |
| 猎人眼 | 精良 | 暴击率+15%，暴击后显示敌人位置2秒 |
| 狂战士之血 | 精良 | 生命低于30%时伤害+50% |
| 死神镰刀 | 稀有 | 击杀有10%概率直接秒杀普通敌人 |
| 时间裂隙 | 稀有 | 撤离等待时间减半，但敌人移速+20% |
| 凤凰之心 | 传说 | 死亡时复活，生命全满，但遗物全部丢失 |
| 诅咒金币 | 传说 | 所有数值×1.5，但无法获得新遗物 |

**遗物槽位：**
- 初始 3 个遗物槽
- 通过升级或特殊事件可扩展至最多 6 个

#### 2.1.4 房间奖励选择

清理房间后，从 3 个奖励中选择 1 个（类似《哈迪斯》）。

**奖励类型：**

| 奖励类型 | 内容 | 出现权重 |
|---------|------|---------|
| 遗物 | 随机遗物 | 30% |
| 强化 | 随机升级选项 | 25% |
| 弹药 | 补充当前武器弹药 | 20% |
| 治疗 | 恢复30%生命 | 15% |
| 金币 | 可在商店购买物品 | 10% |

**奖励稀有度提升：**
- 连续选择同类型奖励，稀有度逐渐提升
- 例如：连续选遗物 → 普通遗物 → 精良遗物 → 稀有遗物

#### 2.1.5 商店系统

地图中随机出现商店房间，用金币购买物品。

**商店物品：**

| 物品 | 价格 | 说明 |
|------|------|------|
| 随机遗物 | 100金币 | 稀有度随机 |
| 生命恢复 | 50金币 | 恢复50HP |
| 武器强化 | 80金币 | 当前武器随机+1属性 |
| 移除诅咒 | 150金币 | 移除1个负面遗物 |

### 2.2 地图系统

#### 2.2.1 地图生成

- 程序生成，房间拼接式
- 每局地图：10-15 个房间（比v2.0更多）
- 地图主题：1个（EA阶段）

#### 2.2.2 房间类型

| 房间类型 | 内容 | 出现权重 |
|---------|------|---------|
| 战斗房间 | 3-6 个敌人 + 奖励选择 | 40% |
| 精英房间 | 1 个精英怪 + 高级奖励 | 15% |
| 商店房间 | 可购买遗物/恢复/强化 | 10% |
| 事件房间 | 随机事件（见事件系统） | 15% |
| 宝藏房间 | 1个遗物（无战斗） | 10% |
| 混合房间 | 敌人 + 物资 | 5% |
| 撤离房间 | 包含撤离点 | 固定在末端 |

#### 2.2.3 房间生成规则

- 第一个房间固定为安全区（选择起始特质后进入）
- 每 4-5 个房间必有一个精英房间
- 最后一个房间固定包含撤离点
- 房间之间通过门连接，开门前显示下个房间类型图标

### 2.3 敌人系统

#### 2.3.1 敌人类型

| 敌人 | 行为特征 | 威胁来源 | 稀有度 |
|------|---------|---------|-------|
| 巡逻兵 | 固定路线巡逻，发现玩家追击 | 数量多 | 普通 |
| 冲锋者 | 高速冲向玩家，近战攻击 | 近战伤害高 | 普通 |
| 狙击手 | 远距离射击，不移动 | 强迫玩家找掩体 | 普通 |
| 投弹手 | 投掷爆炸物，保持距离 | 范围伤害 | 精良 |
| 精英重装兵 | 高血量，中距离射击 | 高血量高伤害 | 精英 |
| 精英刺客 | 隐身接近，高暴击 | 难以发现 | 精英 |
| Boss | 每局末尾出现，多阶段 | 综合威胁 | Boss |

#### 2.3.2 敌人 AI 状态机

```
idle（巡逻）→ alert（发现玩家）→ chase（追击）→ attack（攻击）
                                              ↓
                                          patrol（丢失目标后返回）
```

#### 2.3.3 枪声引来敌人机制

- 打一枪可能引来相邻房间的敌人
- 消音器或近战武器可规避此机制
- 制造战术选择：隐秘行动 vs 火力压制

### 2.4 战斗系统

#### 2.4.1 操作方式

- WASD 移动
- 鼠标瞄准
- 左键射击
- 右键近战（消耗体力）
- 空格冲刺（消耗体力，有冷却）

#### 2.4.2 武器类型

| 武器 | 特点 | 弹药 | 适用build |
|------|------|------|---------|
| 手枪 | 稳定，伤害中等，射速慢 | 9mm | 通用 |
| 霰弹枪 | 高伤害，近距离，射速慢 | 12号霰弹 | 近战流 |
| 冲锋枪 | 射速快，伤害低，弹药消耗快 | 9mm | 火力流 |
| 步枪 | 中距离，高精准，射速慢 | 5.56 | 精准流 |
| 狙击枪 | 远距离，高伤害，极慢射速 | 7.62 | 狙击流 |

#### 2.4.3 武器获取

- 起始特质可能自带武器
- 房间奖励中概率出现
- 商店购买
- 每局最多携带 2 把武器，按数字键切换

### 2.5 撤离系统

#### 2.5.1 撤离流程

1. 到达撤离房间
2. 击败守卫的 Boss（可选，击败有额外奖励）
3. 等待 8-12 秒（可被遗物缩短）
4. 等待期间敌人持续刷新
5. 倒计时结束，成功撤离

#### 2.5.2 撤离结果

| 结果 | 条件 | 后果 |
|------|------|------|
| 成功撤离 | 等待结束 | 保留所有遗物，解锁永久遗物 |
| 死亡 | 等待期间被击杀 | 遗物全部丢失 |
| 强制撤离 | 未到撤离点，时间耗尽 | 只保留1个遗物（自选） |

#### 2.5.3 撤离奖励

| 撤离类型 | 奖励 |
|---------|------|
| 普通撤离 | 保留遗物 |
| 完美撤离（HP>80%） | 额外获得1个随机精良遗物 |
| Boss击杀撤离 | 额外解锁1个永久遗物选项 |

---

### 2.6 永久进度系统（Meta Progression）

#### 2.6.1 永久遗物库

成功撤离时，获得的遗物有概率进入永久遗物库。

- 永久遗物可在后续局中作为起始选项选择
- 每次撤离后随机抽取3个永久遗物，选择1个作为起始遗物
- 鼓励玩家多次游玩，积累遗物库

#### 2.6.2 解锁系统

| 解锁内容 | 条件 |
|---------|------|
| 新特质 | 完成特定成就 |
| 新武器 | 撤离时携带该武器 |
| 新遗物 | 成功撤离时有概率解锁 |
| 新角色外观 | 累计撤离次数 |

#### 2.6.3 统计与成就

- 累计撤离次数
- 累计击杀数
- 最远房间深度
- 各种奇葩死亡方式统计

### 2.7 地图事件系统

#### 2.7.1 随机事件

每个事件房间出现 1 个随机事件，玩家做出选择。

| 事件 | 选项 A | 选项 B |
|------|-------|-------|
| 神秘商人 | 花费50金币购买随机稀有遗物 | 离开 |
| 祭坛 | 献祭20HP，获得精良遗物 | 离开 |
| 受伤的幸存者 | 救治（消耗1治疗道具），获得奖励 | 无视 |
| 诅咒宝箱 | 打开获得稀有遗物，但获得1个负面遗物 | 离开 |
| 赌徒 | 花费30金币，50%概率获得精良遗物，50%概率失去10HP | 离开 |
| 武器大师 | 选择1把武器强化+2 | 选择1把武器换为随机武器 |
| 传送门 | 直接传送到撤离房间，但失去当前30%金币 | 离开 |

#### 2.7.2 环境状态事件

影响整张地图，持续整局。

| 状态 | 效果 | 出现概率 |
|------|------|---------|
| 停电 | 视野范围缩小，敌人也受影响 | 20% |
| 毒气泄漏 | 部分区域持续掉血 | 15% |
| 增援呼叫 | 每隔 90 秒刷新一批新敌人 | 15% |
| 物资充沛 | 所有奖励品质+1级 | 10% |
| 遗物共鸣 | 遗物效果+20% | 10% |

### 2.8 难度系统

#### 2.8.1 热度系统（Heat System，类似《哈迪斯》）

玩家可自主选择增加难度，换取更高奖励。

| 热度等级 | 难度调整 | 奖励提升 |
|---------|---------|---------|
| 0 | 基础难度 | 基础奖励 |
| 1 | 敌人+10%生命 | 金币+10% |
| 2 | 敌人+10%伤害 | 遗物掉落率+5% |
| 3 | 敌人+20%数量 | 金币+20% |
| 4 | 房间数量+3 | 解锁概率+10% |
| 5 | 撤离等待+5秒 | 稀有遗物概率+15% |
| ... | ... | ... |

#### 2.8.2 热度解锁

- 完成首次撤离后解锁热度系统
- 热度等级上限为 20
- 每次更新个人最高热度记录

## 四、技术架构

### 4.1 项目结构

```
Assets/
├── GameCore/
│   ├── GameManager.cs        （全局状态管理）
│   ├── SaveSystem.cs         （JSON序列化存档）
│   └── EventSystem.cs        （模块通信）
```

### 4.2 核心类设计

#### GameManager
- 管理游戏全局状态
- 管理存档读写
- 管理当前局进度

#### RelicManager
- 管理当前局遗物列表
- 计算遗物对属性的加成
- 触发遗物特殊效果

#### TraitSystem
- 管理特质解锁状态
- 应用特质效果
- 处理特质选择

#### LevelSystem
- 管理局内经验值和等级
- 生成升级选项
- 应用升级效果

#### RoomGenerator
- 使用 BSP 算法生成房间布局
- 根据热度调整难度
- 管理房间连接

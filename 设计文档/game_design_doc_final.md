# 逃生指南 · 游戏设计文档

> 独立开发版 | 肉鸽 + 搜打撤
> 最终合并版（整合v5.0+v6.0，删除所有已砍模块）

---

## 一、顶层设计决策

### 1.1 单人。永远单人。

每局控制一个角色，不做队友、不做多人、不做幸存者机制。
撤离失败 = 角色死亡 = 本局结束 = 重开。

### 1.2 三个核心体验目标

活下去 靠装备积累和操作。
逃出去 靠撤离决策和资源管理。
变强得爽 靠Build深度和爆发系统。

### 1.3 设计原则：中期要有爆发感

核心爽感节奏：
初期（第1~3房）：小心翼翼，1~2个遗物，敌人血线尚可
中期（第4~6房）：Build初成型，武器词缀出现，第一次狂暴触发
后期（第7+房）：遗物联动激活，连杀常驻，清理房间从紧张变爽

### 1.4 砍掉的模块（明确不做）

| 模块 | 原因 |
|------|------|
| 多人/队友系统 | 工作量x3，平衡难度爆炸 |
| 钥匙/道具背包 | 背包系统是另一个游戏 |
| 随机抉择事件 | 第一EA版不做，用撤离事件替代 |
| Meta永久进度 | 发EA后再做，现在做等于白做 |
| 热难度曲线（0-20级）| 数值爆炸，用简单版难度递增替代 |
| 侵蚀时间压力 | 反探索，让地图生成白做了 |
| 剧情/叙事 | 像素恐怖不需要剧本 |
| 音效/BGM | 上EA前外包或买资产 |
| 成就/统计系统 | 上EA后再加 |

## 三、局前系统：初始遗物

### 3.1 设计原则

局前遗物决定本局 Build 基调，不决定强度。不选 = 白板，可裸开。

### 3.2 初始遗物池（EA阶段共7个）

|--------|------|---------|
| 武器大师 | 手枪全程可用，无需切换 | 全能型 |
| 疾跑者 | 移速+40%，闪避冷却-30% | 游击型 |
| 弹药狂人 | 弹药掉率+50%，弹匣+2 | 火力型 |

## 四、地图生成系统

### 4.1 地图规模（EA阶段）

- 地图尺寸：固定 64x48 格（1格=1米）
- 房间数量：每局 8~12 个房间（含起始房和撤离房）
- 走廊宽度：固定 2 格
- 房间尺寸：最小 8x8 格，最大 20x16 格

### 4.2 BSP算法

分裂逻辑：
1. 地图初始为整块矩形区域
2. 递归分裂，直到每块区域的最小边长 < 要求的最小房间尺寸
3. 分裂方向：width > height x 1.25 则竖切；height > width x 1.25 则横切；否则随机
4. 分裂位置：较长边的 40%~60% 处随机取

叶节点 -> 房间：每个叶节点生成矩形房间，尺寸 = 叶节点尺寸 x (0.5~0.7)，随机偏移，确保至少2格墙厚。

连接走廊：每对相邻叶节点之间生成L型走廊，走廊穿墙处生成1格宽的门。

连通性验证：生成后用 Flood Fill 从起始房验证全房间可达，不可达的房强制生成额外走廊。

### 4.3 房间类型分配

| 房间类型 | 数量/局 | 说明 |
|---------|---------|------|
| 起始房 | 1 | 固定在地图左下角，无敌人 |
| 撤离房 | 1 | 固定在地图右上角 |
| 战斗房 | 5~8 | 刷怪，清理后开箱 |
| 精英房 | 1~2 | 单个精英敌人，可能掉武器 |
| 商店房 | 1 | 消耗金币购买/重抽 |

### 4.4 难度递增规则

| 已清理房间数 | 敌人HP倍率 | 敌人伤害倍率 | 每房敌人数 |
|------------|------------|------------|---------|
| 0~2 | 1.0 | 1.0 | 2~3 |
| 3~5 | 1.25 | 1.15 | 3~4 |
| 6~7 | 1.5 | 1.3 | 4~5 |
| 8+ | 1.8 | 1.5 | 5~6 |

打得越深，怪越强。

## 五、敌人系统（EA阶段共5种）

### 5.1 设计原则

EA阶段不做AI行为树。用有限状态机（FSM）+ 数值差异来区分敌人。

### 5.2 五种敌人

| 敌人 | 类型 | HP倍率 | 行为 | 克制方式 |
|------|------|--------|------|---------|
| 巡逻兵 | 普通 | x1.0 | 固定路线巡逻，看到玩家后追击 | 绕背/引开 |
| 冲锋兵 | 普通 | x0.8 | 每8秒直线冲刺，冲刺时无敌 | 冲刺前后摇，绕侧打 |
| 投掷兵 | 普通 | x0.7 | 每6秒向玩家投掷手雷（范围伤害）| 优先击杀/走位躲 |
| 精英枪手 | 精英 | x3.0 | 中距精准射击，弹匣大，2秒换弹 | 利用换弹窗口输出 |
| Boss：持盾巨人 | Boss | x8.0 | 缓慢移动，正面盾挡，间歇性震地波 | 绕后/引震地波后打侧背 |

HP倍率 = 相对基准敌人HP（100）的倍数。

### 5.3 状态机

idle -> alert（发现玩家，原地愣1秒）-> chase -> attack -> patrol（丢失目标5秒后返回）

感知范围：巡逻兵5格 / 冲锋兵7格 / 投掷兵8格 / 精英枪手12格 / Boss10格

### 5.4 刷怪规则

玩家进入房间 -> 关门 -> 2秒后敌人从预设位置生成 -> 全部击杀 -> 门解锁 -> 掉落箱子。

## 六、战斗系统

### 6.1 操作

| 输入 | 动作 |
|------|------|
| WASD | 移动 |
| 鼠标移动 | 瞄准方向 |
| 左键 | 射击 |
| 右键 | 切换武器 |
| 空格 | 闪避（翻滚，0.3秒无敌帧，冷却8秒）|
| R | 换弹 |

### 6.2 武器基础属性（EA阶段共5把）

| 武器 | 射速(发/秒) | 单发伤害 | 弹匣 | 换弹(秒) | 特点 |
|------|------------|---------|------|---------|------|
| 手枪 | 3 | 25 | 12 | 1.5 | 稳定，可移动射击 |
| 冲锋枪 | 9 | 12 | 30 | 2.0 | 高射速，低单发，弹道略散 |
| 霰弹枪 | 1.2 | 15x6(散) | 6 | 2.5 | 近距AOE，扩散后伤害递减 |
| 突击步枪 | 5 | 30 | 25 | 2.2 | 均衡，中距精准 |
| 狙击枪 | 0.6 | 150 | 5 | 3.0 | 极高伤害，低射速 |

弹药获取：敌人掉落弹药盒，每盒给当前武器15~25发，狙击枪给5发。

### 6.3 武器变体词缀系统

每把武器被击杀敌人时，有概率获得一个变体词缀（每把武器最多1个词缀）。
词缀替换武器的部分行为或数值，是每次体验不同的核心来源。

词缀池（EA阶段共8种）：

| 词缀名 | 触发条件 | 效果 | 适合武器 |
|--------|---------|------|---------|
| 贯穿 | 击杀时 | 子弹穿透1个敌人，伤害-20% | 手枪/步枪 |
| 爆发 | 击杀时 | 击杀触发小范围爆炸（20范围伤害）| 冲锋枪/霰弹 |
| 燃烧 | 每3次命中 | 目标附加灼烧，每秒5伤害，持续3秒 | 步枪/狙击 |
| 吸血 | 击杀时 | 回复造成伤害的30%作为HP | 手枪/冲锋枪 |
| 连锁 | 命中时 | 子弹弹跳至最近敌人，最多1次，伤害-30% | 手枪/冲锋枪 |
| 冰冻 | 每5次命中 | 目标冻结0.5秒 | 霰弹/步枪 |
| 暴击流 | 击杀时 | 暴击率+15%（全局），持续5秒，可叠加 | 狙击/步枪 |
| 无尽 | 击杀时 | 额外回复当前武器5发弹药 | 霰弹/冲锋枪 |

同一把冲锋枪，这次是"贯穿+吸血"，下次可能是"爆发+无尽"。词缀组合决定战斗节奏。

### 6.4 武器掉落

精英敌人：必定掉落1把带词缀的武器
Boss敌人：掉落1把带词缀武器 + 3个遗物选项（必定史诗或以上）

武器掉落后自动进入武器背包（初始手枪+最多2把额外武器），右键切换无冷却。

### 6.5 数值基准

| 属性 | 数值 |
|------|------|
| 玩家最大HP | 100 |
| 玩家基础移速 | 5格/秒 |
| 闪避无敌帧 | 0.3秒 |
| 闪避冷却 | 8秒 |
| 基准敌人HP | 100（xHP倍率）|
| 基准敌人伤害 | 20/次（x伤害倍率）|
| 暴击基准倍率 | x1.5 |
| 暴击基准率 | 5% |
| 移动射击散布 | +15% |

## 七、爆发系统：狂暴 + 连杀

这是游戏爽感的核心来源。狂暴提供持续性的强度曲线，连杀提供间歇性的爆发峰值。

### 7.1 狂暴系统（Rage）

触发方式：每次击杀敌人积累狂暴值，房间内连续击杀持续上升。

| 狂暴等级 | 积累值 | 触发效果 | 持续时间 | 冷却时间 |
|---------|--------|---------|---------|---------|
| 狂暴 I | 50 | 攻速+25%，移速+15% | 直到本房清理完 | 无 |
| 狂暴 II | 150 | 攻速+50%，移速+25%，狂暴II特效激活 | 直到本房清理完 | 无 |
| 狂暴 III（爆发）| 300 | 全屏AOE冲击（每5秒一次，50范围伤害），攻速+75% | 15秒 | 30秒 |

积累规则：
- 每个普通敌人：+10狂暴值
- 每个精英敌人：+50狂暴值
- 每次清房：狂暴值不清零，跨房间累积
- 玩家受伤：狂暴值-20
- 狂暴III触发后进入冷却，冷却期间狂暴值保持在200（不归零）

狂暴III的AOE效果是全局的，让后期清理房间有割草感——但这是Build结果，不是起点。

### 7.2 连杀系统（Kill Streak）

触发方式：连续击杀敌人且间隔不超过4秒，连杀计数累加。

| 连杀数 | 伤害加成 | 说明 |
|-------|---------|------|
| 5连 | 伤害+25% | 首次触发，连杀图标亮起 |
| 10连 | 伤害+50%，狂暴积累x1.5 | 音效/视觉反馈 |
| 20连 | 伤害+100%，暴击率+20% | 屏幕边缘发光 |
| 30+连 | 每+10连，伤害+25% | 无上限，清房后持续 |

断连规则：4秒内未击杀敌人，连杀归零。
记忆机制：历史最高连杀每局只记录一次，超过历史最高时给予额外狂暴值+20。

### 7.3 双系统联动

狂暴系统提供持续性的强度曲线（越打越强），
连杀系统提供间歇性的爆发峰值（连杀断档会失去加成）。
两者互相拉扯：想维持连杀必须快打（激进），想积累狂暴必须不停杀（贪心）。

|------|---------|

| 无尽弹药 + 霰弹聚焦 | 霰弹每次12颗弹丸（原6颗），换弹-30% |
| 死亡标记 + 暴击回路 | 暴击击杀爆炸伤害x2，爆炸范围+50% |
| 疾跑者 + 双重武装 | 换弹时获得0.5秒无敌，移速额外+15% |
| 毁灭打击 + 穿甲弹 | 强化射击自动穿透所有命中敌人，伤害不递减 |
| 硬汉 + 硬甲板 | 受伤硬直完全消除，且受伤后0.5秒内反击伤害+50% |

### 8.4 特性池（EA阶段共16个）

普通特性（清理房间时随机1个，选1）：
火力增强/迅捷/硬甲/弹药储备/稳枪（各+5%~+15%）

稀有特性（精英击杀时必出1个，选1）：
穿透射击/暴击精通/弹药狂潮/战术换弹/坚韧

史诗特性（Boss击杀时必出2个，选1）：
- 战意沸腾（HP<30%时伤害+60%）
- 双重射击（同时开火两把枪）
- 复仇（死亡时对周围敌人造成50范围伤害，一次）

### 8.5 特性重复升级

同一个特性再次获得时，升级强化（不是浪费）：

| 特性 | 第一次 | 第二次（升级I）| 第三次（升级II）|
|------|-------|---------------|---------------|
| 火力增强 | +10%伤害 | +25%伤害 | +45%伤害 |
| 迅捷 | +15%移速 | +30%移速 | +50%移速，闪避冷却-1秒 |
| 暴击精通 | +25%暴击率 | +45%暴击率 | +70%暴击率 |
| 弹药狂潮 | 击杀回复15%弹匣 | 击杀回复30%弹匣 | 击杀回复50%弹匣 |

### 8.6 Build 构建思路

遗物决定长期强度和联动潜力，
特性决定即时战斗强化，
武器词缀决定每局核心输出节奏，
连杀+狂暴决定爆发峰值。

## 十、撤离系统

### 10.1 撤离流程

1. 玩家到达撤离房
2. 触发撤离事件（3种随机之一）
3. 等待90秒倒计时（期间敌人从地图各处涌来）
4. 倒计时结束前进撤离圈 -> 撤离成功；倒计时归零未进入 -> 撤离失败

### 10.2 撤离事件（3种随机）

| 事件 | 内容 | 选项 |
|------|------|------|
| 物资补给 | 撤离圈内出现一个免费遗物箱 | 直接领取 |
| 紧急警报 | 立即刷新3只巡逻兵 | 立即开门可跳过战斗直接撤离（损失奖励）|
| 导航员上线 | 倒计时减少15秒，获得导航特性（显示撤离房方向）| 接受/拒绝 |

### 10.3 撤离奖励

成功撤离后随机获得：随机普通遗物 / 随机稀有遗物（20%概率）/ 50金币。

## 十三、开发阶段划分

### 第一阶段：能跑通（目标：第1~6周）

| 任务 | 验收标准 |
|------|---------|
| 玩家移动+射击 | 在空场景里能打靶子 |
| BSP地图生成 | 生成房间+走廊，能走进每个房间 |
| 3种普通敌人 | 巡逻兵/冲锋兵/投掷兵，有FSM，能被打死 |
| 战利品箱子 | 敌人死后掉落箱子，打开出3选项 |
| 房间切换 | 走门进下一个房间，不穿墙 |
| 撤离系统 | 撤离房触发倒计时，能完成一次完整撤离 |

阶段终点：一个人能完成一次"选遗物->走图->战斗->撤离"的完整闭环。

### 第二阶段：核心好玩（目标：第7~18周）

- 5把武器完整数值（手感/后座力/弹道调好）
- 武器词缀系统（8种词缀全部实装 + 触发检测）
- 狂暴系统（积累条/等级触发/全屏AOE + 视觉反馈）
- 连杀系统（计数/计时/伤害加成应用）
- 遗物联动检测（6组联动全部实装 + 触发特效）
- 特性重复升级（升级逻辑全部实装）
- 遗物系统完整（20个遗物全部实装）
- 特性系统完整（16个特性全部实装）
- 精英敌人（精英枪手）
- Boss战（持盾巨人）
- 武器掉落（精英/Boss掉落带词缀武器）
- 商店重抽（100金币重随机）
- UI完善（HUD/狂暴条/连杀计数器/遗物图标）
- 数值平衡（第一版通关率目标：30%~50%）

阶段终点：每局有明显的Build差异感，中期狂暴+连杀的爆发感强烈，能稳定通关。

### 第三阶段：准备EA（目标：第19~26周）

- 关卡美术（统一像素风格，地图视觉不丑）
- 敌人美术（5种敌人各一套像素动画）
- 玩家动画（移动/射击/闪避/死亡）
- 粒子特效（枪口火焰/命中血花/爆炸/狂暴AOE/连杀特效）
- UI美术（开始菜单/暂停/HUD/狂暴条特效/连杀特效/结算）
- Steam商店页（截图+描述+预告片）
- 60秒预告片（展示核心玩法+Build多样性）
- Bug清理（完整测试一轮，修崩溃）

阶段终点：在Steam发布EA版，定价25元。

## 十五、一句话版本

玩家操控一个士兵，在随机生成的建筑里用不同武器和遗物构建Build，
清光房间获取战利品，找到撤离点活着出去。
Build越强越容易撤，build越差越要赌。
而狂暴+连杀的双爆发系统，让"变强"从线性变成指数。
